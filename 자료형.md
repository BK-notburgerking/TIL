# 자료형



## 기본 자료형

- 데이터가 변수에 직접 저장
- 메모리에 데이터가 직접 들어감

### 정수형

	- byte : 1byte
	- __char__ : 2byte
	- short : 2byte
	- __int__ : 4byte
	- long : 8byte

### 실수형

- float : 4byte
- __double__ : 8byte

### 논리형

- __boolean__ : 1byte



## 객체 자료형

- `String` 은 객체자료형임

- 객체 메모리 주소가 변수에 저장
- 메모리에 객체가 존재
- 객체를 가르키는 다른 메모리의 공간을 확보
  - 이것은 객체가 존재하는 메모리의 주소값을 가지고 있음
  - 주소값만을 갖고 있기 때문에 __4byte__로 고정
- C에서의 포인터
- Java에선 __레퍼런스__라고 함
